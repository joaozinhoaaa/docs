# API Promisse - Gateway de Pagamento

Esta API permite gerenciar pagamentos online de forma simples e segura.

## Autenticação

Todos os endpoints requerem Bearer Token no header `Authorization`.

## Endpoints disponíveis

- **GET** `/payments` — Lista pagamentos
- **POST** `/payments` — Cria pagamento
- **DELETE** `/payments/{id}` — Remove pagamento
- **POST** `/payments/webhook` — Webhook de eventos
